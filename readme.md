# Binary Text Classification

This project is a simple machine learning model that classifies text into two categories. I built it while learning NLP and wanted to understand how real-world text classification systems work.

---

## What this project does

- Takes text input  
- Cleans and processes it  
- Converts text into numerical form using TF-IDF  
- Predicts the label using a trained model  

---

## Tech used

- Python  
- Pandas, NumPy  
- Scikit-learn  
- Jupyter Notebook  

---

## How it works

1. Load the dataset  
2. Clean the text (lowercase, remove missing values)  
3. Split into train and test data  
4. Convert text using TF-IDF vectorizer  
5. Train a Logistic Regression model  
6. Evaluate using classification report and confusion matrix
